## Install

`npm i -g @ahkui/curseforge-mods-manager`



## Usage
### Show installed mod
`cmm show`

### Add a new mod
`cmm add`

### Update exist mods
`cmm update`

### Sync with server modlist
`cmm sync`

### Sync with specify server modlist
`cmm sync --server https://example.com/cmm_config.json`

### Sync and download all mods
`cmm sync -f`

### Download all mods
`cmm fetch`
