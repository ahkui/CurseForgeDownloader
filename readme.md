## Install

`npm i -g curseforge-downloader`



## Usage
### Show installed mod
`cfd show`

### Add a new mod
`cfd add`

### Update exist mods
`cfd update`

### Sync with server modlist
`cfd sync`

### Sync with specify server modlist
`cfd sync --server https://example.com/cfd_mod_config.json`

### Download all mods
`cfd fetch`
